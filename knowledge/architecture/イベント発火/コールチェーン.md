## イベント発火 コールチェーン

おさわり後の `EventEvaluator.Evaluate` 起点で、シナリオ実行まで4つのフェーズで処理される。
フェーズ1〜3は `EventEvaluator.Evaluate` の中で完結し、フェーズ3の `OnEventPublished` イベントがフェーズ4（Unity側）へバトンを渡す。

---

### フェーズ1: イベント候補の収集

```
EventEvaluator.Evaluate(Interaction)   // core/.../Application/EventEvaluator.cs:29
  ├→ interactionHistory.Record(interaction)        // タッチ履歴に記録（フェーズ2の条件評価で参照される）
  ├→ eventRepository.GetEvents(PartID)             // 触れた部位のイベント
  └→ eventRepository.GetGlobalEvents()             // どの部位でも対象のイベント
```

収集した partEvents + globalEvents を結合してフェーズ2へ渡す。

---

### フェーズ2: 条件評価・選択

収集したイベント全件に対して `CanFire()` を評価し、発火するイベントを1つ選ぶ。

```
Event.CanFire()                        // core/.../Domain/Events/Entity/Event.cs
  └→ ICondition[].CanFire() を AND評価（1つでも false なら不発）
```

| Condition             | 判定内容                                             | シナリオキー    |
| --------------------- | ---------------------------------------------------- | --------------- |
| EventFiredCondition   | 指定イベントが過去に発火済み                         | `event_fired`   |
| FirstTouchCondition   | 対象部位への初回タッチ (count==1)                    | `first_touch`   |
| TouchCountCondition   | タッチ回数 >= minCount                               | `touch_count`   |
| ThresholdCondition    | パラメータ値が範囲内 (affection/arousal/development) | `affection` 等  |
| HeroinClimaxCondition | Heroin.IsClimax == true                              | `heroin:climax` |

条件を満たすイベントが複数ある場合、Priority 最大のものが1つ選ばれてフェーズ3へ渡る。
条件を満たすイベントがなければここで終了（シナリオ起動なし）。

---

### フェーズ3: 発火・ログ記録

選ばれたイベントを `EventBus.Publish` に渡す（まだ `EventEvaluator.Evaluate` の中）。

```
EventBus.Publish(fired)                // core/.../Infrastructure/EventBus/EventBus.cs
  ├→ firedEventLog.Record(eventID)    // 発火済みログに記録（次回以降 EventFiredCondition で参照）
  └→ OnEventPublished?.Invoke(fired)  // ← ここでフェーズ4（Unity側）へ渡る
```

---

### フェーズ4: シナリオ開始

`OnEventPublished` を購読している `ScenarioDriver` が受け取り、シナリオを起動する。

```
ScenarioDriver.OnEventFired(event)     // unity/Assets/Scripts/Presentation/ScenarioDriver.cs
  ├─ Priority >= 900 && 再生中 → Stop() + Play()   // 割り込み発火
  ├─ !再生中               → Play()               // 通常発火
  └─ 再生中                → 無視                  // パラメータ更新は完了済み

ScenarioPlayer.Play(ScenarioID)        // unity/Assets/Scripts/Presentation/ScenarioPlayer.cs
  └→ YarnScenarioEngine.Start(scenarioID)
        └→ DialogueRunner.StartDialogue(scenarioID)  // Yarn Spinner 実行開始
```
