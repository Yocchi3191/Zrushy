## イラスト差し替え コールチェーン

ボタン入力からスプライト更新までの完全なコールチェーン:

```
[Unity/PoC or Scenario]
PoCButton.OnClick()                                        // unity/Assets/PoC/ChangeIllust/PoCButton.cs
  → SpriteLayerController.ChangeSprite(SpriteLayerID, LayerState)
                                                           // core/Zrushy.Core.Presentation/SpriteLayerController.cs
      → ChangeSprite.Execute(SpriteLayerID, LayerState)   // core/Zrushy.Core.Application/UseCase/ChangeSprite/ChangeSprite.cs
            → ISpriteLayerRepository.Get(SpriteLayerID, LayerState)
                  return "Heroin/{layerID}/{state}"        // unity/Assets/Scripts/Infrastructure/SpriteLayer/SpriteLayerRepository.cs
      → HeroinViewModel.UpdateSprite(SpriteLayerID, string)
                                                           // core/Zrushy.Core.Presentation/HeroinViewModel.cs
            → OnUpdated?.Invoke(this)

[Unity/Presentation]
HeroinView.OnViewModelUpdated(HeroinViewModel)             // unity/Assets/Scripts/Presentation/HeroinView.cs
  → HeroinView.UpdateSpriteLayers(Dictionary<SpriteLayerID, string>)
      → bindings から SpriteLayerID に対応する Image を検索
      → path が "/none" で終わる場合: Image.enabled = false
      → それ以外: Resources.Load<Sprite>(path) → Image.sprite = sprite
```
