## 服ずらし反応 コールチェーン

ドラッグ入力からシナリオ再生までの完全なコールチェーン。おさわりと並行する構造。

```
[Unity/Presentation]
Zrushable.OnEndDrag(PointerEventData)
  → IZrushyClothing.Execute(ZrushyInput) : ZrushyResult
       → Heroin.CanPutOffClothing(ClothingID) → bool
       → ZrushyResult = bool ? Success : Failure
       → IClothingEventEvaluator.Evaluate(ClothingID, ZrushyResult)
            → IZrushyHistory.Record(ClothingID, ZrushyResult)
            → IClothingEventRepository.GetEvents(ClothingID, ZrushyResult)
            → IScenarioEvent.CanFire() フィルタ → Priority ソート
            → EventBus.Publish(fired)
       → return ZrushyResult
  → result == Success → _spriteInputHandler.TryTransition(input)
```

EventBus.Publish 以降はおさわりと同じ導線（ScenarioPlayer → Yarn）。
詳細: `knowledge/architecture/イベント発火/コールチェーン.md`

## おさわりとの対比

| 役割 | おさわり | 服ずらし |
|---|---|---|
| Presentation | `Clickable` | `Zrushable` |
| UseCase | `InteractPart` | `ZrushyClothing` |
| Evaluator I/F | `IEventEvaluator` | `IClothingEventEvaluator` |
| Evaluator 実装 | `EventEvaluator` | `ClothingEventEvaluator` |
| Repository I/F | `IEventRepository` | `IClothingEventRepository` |
| History | `IInteractionHistory` | `IZrushyHistory` |
| 発火キー | `PartID` | `ClothingID × ZrushyResult` |

## 新規コンポーネント

### Domain

| クラス/インターフェース | 配置 | 概要 |
|---|---|---|
| `ZrushyResult` | `Domain/Interactions/ValueObject/` | `Success` / `Failure` の enum |
| `IClothingEventEvaluator` | `Domain/Events/Service/` | `Evaluate(ClothingID, ZrushyResult)` |
| `IClothingEventRepository` | `Domain/Events/Repository/` | `GetEvents(ClothingID, ZrushyResult)` |
| `IZrushyHistory` | `Domain/Interactions/Repository/` | `Record(ClothingID, ZrushyResult)` ／将来の Condition 用 |

### Application

| クラス | 配置 | 概要 |
|---|---|---|
| `ClothingEventEvaluator` | `Application/` | `IClothingEventEvaluator` 実装 |

### 変更

| クラス | 変更内容 |
|---|---|
| `IZrushyClothing` | `CanZrushy(input): bool` → `Execute(input): ZrushyResult` |
| `ZrushyClothing` | `IClothingEventEvaluator` を注入。判定後に評価を呼ぶ |
| `Zrushable` | `Execute` の戻り値が `Success` のときのみ `TryTransition` を呼ぶ |
